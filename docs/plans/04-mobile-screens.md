# 04. 모바일 앱 화면 설계 (Mobile Screen Design)

## 📱 개요
청소 예약 시스템의 모바일 앱(Expo React Native) 화면 구조와 주요 기능을 정의합니다. 요청자(Customer)와 제공자(Cleaner) 역할에 따른 사용자 흐름을 중심으로 설계되었습니다.

---

## 🎨 공통 UI 요소
- **네비게이션**: Expo Router 기반 탭 네비게이션 및 스택 네비게이션
- **테마**: NativeWind(Tailwind CSS)를 활용한 화이트/블루 테마
- **인증**: 소셜 로그인(카카오, 네이버) 연동

---

## 👨‍💻 요청자(Customer) 화면

### 1. 홈 (Home)
- **목적**: 현재 예약 상태 요약 및 신규 요청 버튼
- **기능**:
  - 현재 진행 중인 예약(확정됨) 카드 노출
  - '새로운 청소 요청하기' 플로팅 버튼 (FAB)
  - 최근 이용한 청소 매니저 목록 (선택 사항)

### 2. 청소 요청하기 (Create Booking)
- **목적**: 단계별 폼을 통한 청소 예약 생성
- **기능**:
  - **1단계**: 서비스 지역 선택 (행정동 단위)
  - **2단계**: 날짜 및 시간 선택 (캘린더 UI)
  - **3단계**: 주소지 상세 정보 및 요구사항 입력 (반려동물 여부, 특이사항 등)
  - **4단계**: 예상 견적 확인 및 요청 완료

### 3. 내 예약 목록 (My Bookings)
- **목적**: 전체 예약 이력 관리
- **기능**:
  - 탭 구분: 대기중 / 확정됨 / 완료 / 취소
  - 각 항목 클릭 시 상세 화면 이동

### 4. 예약 상세 및 채팅 (Booking Detail & Chat)
- **목적**: 특정 예약의 세부 정보 확인 및 제공자와의 소통
- **기능**:
  - 예약 정보 (시간, 장소, 금액) 요약
  - 제공자 프로필 링크
  - 예약 기반 1:1 메시지 (폴링 방식)
  - 청소 완료 후 '리뷰 작성' 버튼 활성화

### 5. 마이페이지 (Profile)
- **목적**: 개인 정보 및 환경 설정
- **기능**:
  - 내 정보 수정 (연락처, 기본 주소)
  - 알림 설정
  - 고객센터 및 약관 확인

---

## 🧹 제공자(Cleaner) 화면

### 1. 매니저 대시보드 (Cleaner Dashboard)
- **목적**: 오늘 할 일 및 신규 요청 확인
- **기능**:
  - 오늘 예정된 청소 일정 (시간순 리스트)
  - 실시간 수락 대기 중인 요청 알림 배지
  - 이번 달 수익 현황 요약

### 2. 요청 찾기 (Find Requests)
- **목적**: 수락 가능한 주변 청소 요청 탐색
- **기능**:
  - 서비스 가능 지역 내의 '대기 중' 요청 리스트
  - 거리순/시간순 정렬 필터
  - 상세 확인 후 '수락' 버튼

### 3. 일정 관리 (Schedule)
- **목적**: 주간/월간 일정 및 활동 시간 설정
- **기능**:
  - 서비스 가능 시간대 설정 (요일별 가능/불가능)
  - 캘린더 형태의 예약 현황 조회

### 4. 메시지 함 (Messages)
- **목적**: 진행 중인 모든 고객과의 대화 목록
- **기능**:
  - 예약 건별 메시지 채널 리스트
  - 읽지 않은 메시지 표시

### 5. 매니저 프로필 (Cleaner Profile)
- **목적**: 고객에게 보여지는 전문성 관리
- **기능**:
  - 자기소개 및 경력 수정
  - 서비스 지역 설정
  - 평점 및 리뷰 모아보기

---

## 🔄 주요 사용자 흐름 (User Flows)

### [예약 프로세스]
1. **요청자**: 홈 → 요청 등록 → 대기
2. **제공자**: 요청 찾기 → 상세 확인 → 수락
3. **시스템**: 상태를 `WAITING`에서 `CONFIRMED`로 변경 → 양측 알림
4. **소통**: 양측 예약 상세 화면에서 메시지 교환

### [완료 및 리뷰]
1. **제공자**: 청소 완료 후 '완료 처리' 버튼 클릭
2. **시스템**: 상태를 `COMPLETED`로 변경
3. **요청자**: 리뷰 작성 알림 수신 → 별점 및 텍스트 리뷰 등록
4. **제공자**: 고객에 대한 매너 평가 등록
